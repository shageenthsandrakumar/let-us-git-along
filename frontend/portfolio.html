How it's built

FounderFit uses ag2's group chat infrastructure to run a coordinated multi-agent pipeline. Each agent is a ConversableAgent with a structured system prompt that enforces JSON output. The AutoPattern routes messages between agents automatically based on context.

Agent Framework

ag2 (formerly AutoGen) ConversableAgent with AutoPattern group chat coordination.

ag2[openai]

LLM Routing

OpenRouter provides access to GPT-4o with automatic fallback and rate limit handling.
